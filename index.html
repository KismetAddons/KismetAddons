How to install the mod?

The mod needs ChatTriggers (can be downloaded from https://www.chattriggers.com) to work. After it is installed and put in your mods folder. you should start minecraft and type /ct files in the chat and put the downloaded mods in the modules section and overwrite files if needed. After that type /ct reload in chat and the mod should start working (no restart needed)
